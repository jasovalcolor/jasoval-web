Sobre la carta RAL

La carta RAL Classic es el sistema de codificación de colores más utilizado en Europa para industria, arquitectura y señalización. Cada color tiene un código de 4 dígitos (del 1000 al 9018) y un nombre normalizado. Es el estándar de referencia para especificar colores en pedidos industriales.

Sí. En Jasoval Color fabricamos bajo pedido en todos los colores RAL Classic. Utilizamos pastas pigmentarias de alta calidad compatibles con nuestros gel coats, esmaltes y resinas para reproducir con fidelidad cualquier tono de la carta RAL.

Los colores más habituales (blanco, gris, negro, azul tráfico, rojo tráfico, verde…) suelen estar disponibles en 24–48 horas. Para colores menos frecuentes el plazo habitual es de 3 a 5 días laborables. Consúltanos para confirmar disponibilidad.

Las representaciones digitales son aproximaciones, ya que cada monitor reproduce los colores de forma distinta. Para validar el color exacto recomendamos solicitar una muestra física antes de un pedido de producción.

Sí, ofrecemos muestras físicas de color para que puedas validar el tono antes de confirmar la producción. Contáctanos indicando el código RAL y el producto que necesitas.
